# hyprconfig
My Simple Hyprland Config  

This setup is intentionally minimal and focused on work.

Most utilities are implemented separately using custom scripts (Bash/Python) and run in the background, rather than relying on additional packages.

Please read the documentation carefully before using it, as the workflow and keybindings may take some time to get used to.

# Information
OS: ```Arch Linux```

PKG: ```pacman```

WM: ```Hyprland```

Terminal: ```alacritty``` ```kitty```

Editor" ```neovim```

StatusBar: ```waybar```

Notification: ```swaync```

Shellscript: ```bash``` ```zsh```

Launcher: ```rofi```
